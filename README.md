# wallpainter

Download images as desktop wallpapers.

## Development

Server:

```sh
$ pipenv install
$ pipenv run start
```

Client:

```sh
$ cd app

# Development web page
$ yarn
$ yarn dev:web

# Open in electron
$ yarn dev
```

## Start

```sh
$ cd app

# Build static files
$ yarn build

# Start electron
$ yarn start
```
