# wallpainter

Download images as desktop wallpapers.

## Development

Server:

```sh
$ pip install -r requirements.txt
$ python -m wallpainter
```

Client:

```sh
$ cd app

# Development web page
$ yarn
$ yarn dev:web

# Open in electron
$ yarn dev
```

## Start

```sh
$ cd app

# Build static files
$ yarn build

# Start electron
$ yarn start
```
